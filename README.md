# Netflix-Data-Analysis# Netflix Content Analysis Dashboard

## Project Overview
Interactive Tableau dashboard analyzing Netflix movies and TV shows to uncover trends in content type, ratings, genres, and release patterns.


## Tools Used
- Tableau Public
- Python
- Pandas

## Key Insights
- Movies contribute ~70% of Netflix catalog
- TV-MA is the most common rating
- Drama & International genres dominate content categories
- Content additions increased significantly after 2015
- Most titles have runtimes between 80–120 minutes
